# Зикр

Ўзбек тилида (кирилл) зикр ва дуолар тўплами. Ягона статик HTML файл.

## Тузилиш

- `index.html` — бутун лойиҳа (HTML + inline CSS + inline JS). Build йўқ.
- `favicon.svg`, `favicon.png`, `apple-touch-icon.png` — иконкалар.
- Бошқа файл йўқ.

## Хостинг

GitHub Pages, `main` бранчдан хизмат қилади. Merge-дан 1–2 дақиқа кейин жонли.

## Workflow

- `main` бранч protected — тўғридан-тўғри push мумкин эмас.
- Ишни `claude/review-and-feedback-W7BYo` (ёки бошқа feature) бранчда қилиш.
- Commit → push → PR очиш → фойдаланувчи GitHub-да merge қилади.

## Контент тузилиши

Беш гуруҳ (`.group` → `.group-header` + `.group-body`):
1. Дуо боши ва охирида айтиладиган илтижолар
2. Эрталабки зикрлар
3. Кечқурунги зикрлар
4. Жума куни дуолари
5. Руқия

Ҳар бирида `.card` элементлар. Кўпинча `.card-arabic` (арабча матн) + `.card-footer` (`.card-count` манба/такрор сони). Жума ва дуо боши гуруҳларида қўшимча `.card-number` бор.

Бир оят/дуо мантиқан бир бутун бўлса — битта картага қўйилади (масалан Наҳл: 68-69). Кетма-кет оятлар орасига `۝` ажратгичи қўйилади.

## Дизайн қарорлари

- Палитра CSS variables-да (`:root` ва `[data-theme="dark"]`).
- Шрифтлар: Cormorant Garamond (logo, рақамлар), Scheherazade New (арабча), PT Serif (асосий матн).
- Theme: `localStorage`-да, FOUC олдини олиш учун inline скрипт `<head>`-да эмас, `<body>` бошида.
- Шрифт катталаштириш А+/А− — фақат арабча матнга таъсир қилади (`--base-font-size` орқали).

## Қоидалар

- Эмодзи қўшмаслик (ҳеч қаерда — UI, commit, доc).
- Ўлик код қўшмаслик: декларация ишлатилмаса, ё чақириш, ё ўчириш.
- Янги хусусият қўшмасдан олдин фойдаланувчидан сўраш.

## Тил

Фойдаланувчи ўзбек кирилл ёзади. Жавоблар ҳам ўзбек кирилл бўлсин. Commit message ва код изоҳлари инглизча бўлиши мумкин (стандарт амалиёт).
